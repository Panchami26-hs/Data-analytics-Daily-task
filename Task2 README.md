Objective

Create meaningful visualizations using Power BI that tell a compelling story based on business insights from the Superstore sales dataset.


Power BI Dashboard (PBIX file)

 Tools Used

Power BI Desktop

Superstore Sales Dataset

Steps Completed
1. Data Import

Imported the Superstore.csv dataset into Power BI.

Checked data types and corrected where necessary.

2. Data Cleaning

Removed duplicates.

Corrected data types (Date, Numeric, Text).

Created new calculated fields (Profit %, Sales Category if applicable).

3. Data Analysis

Identified key insights:

Copiers and Phones are highly profitable

Tables are showing losses

Categories/Subcategories performance analyzed

Regional sales/ profit trends evaluated

Customer/ship mode patterns observed

4. Visualizations Created

The dashboard includes visuals such as:

Bar Chart: Sales & Profit by Category

Donut Chart: Profit Contribution %

Map: Sales by Region

Line Chart: Monthly Sales Trend

KPI Cards:

Total Sales

Total Profit
Total Orders
